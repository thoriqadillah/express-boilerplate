# Express boilerplate backend

## Development
- Install the dependency 
```bash
npm i
```

- Copy the `.env.example` to `.env`

### Database schema type generation
Generate the table type automatically
```bash
npx kysely-codegen
```

### Run the development server
```bash
npm run dev
```

## Build
Build the project
```bash
npm run build
```

### Run the built server
```bash
npm run start
```

## Testing
```bash
npm run test
```
